GAME RULES

Roles

  • Babu: Immune player, always gets 100 points
  • Pulish: The guesser - must find the culprit
  • Chor: Thief - target in Chor rounds
  • Dakat: Bandit - target in Dakat rounds

Round Types

  • Chor Round: Pulish must find the Chor
  • Dakat Round: Pulish must find the Dakat

Rounds alternate: Chor, Dakat, Chor, Dakat...

Scoring

  • Babu: Always 100 points
  • Pulish: 80 points if correct, 0 if wrong
  • Target Culprit: 0 if caught, 40/60 if escapes
  • Non-target Culprit: Always immune

How to Play

  1. Each player views their role secretly
  2. Babu and Pulish are revealed
  3. Pulish has 30 seconds to guess
  4. Points are awarded based on result
  5. Game continues for 10 rounds
